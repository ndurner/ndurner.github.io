Open Responses is a new specification inspired by the OpenAI Responses API for interacting with LLMs/VLMs/LRMs. The launch partners include OpenAI, OpenRouter, Hugging Face, ollama, vLLM, LM Studio, and Vercel. It could thus supersede the legacy Chat Completions API. The specification is not all encompassing, however. One pitfall I have recently experienced was around how to include reasoning items.
