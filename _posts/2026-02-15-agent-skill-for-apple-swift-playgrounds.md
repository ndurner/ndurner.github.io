OpenAI’s LLMs have improved utility in Swift coding a lot since GPT-4. But still, the [Apple Swift Playground(s)](https://developer.apple.com/swift-playground/) variety - particularly on iPad - is not covered well. (Swift Playgrounds is positioned for programming beginners, but another plus is that it’s a dynamic code execution environment on iPad). With a recent EPUB creation project from online sources, the process degraded into the familiar back and forth with the AI implementing possible remedies to errors and me feeding iPad screenshots of new errors back to it.

The mistakes were basic ones, for example:

- Manifest mismatches for Playgrounds app packages (.iOSApplication schema issues)
